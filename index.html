Ty's Tic Tac Toe Game

Instructions

These game requires two players. Player 1 will be "X" and Player 2 will be "O". When player 1 clicks on a square an "X" will appear and the same goes for Player 2. You take turns. The objective of the game is to get 3 "X's" or "O's" (depending on which player you are) in a row. The Screen will flash when the game is over!
